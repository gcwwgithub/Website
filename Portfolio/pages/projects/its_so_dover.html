CheA+ing Simulator (2024)
Role: Gameplay Programmer
Team Size: 6
Project Length: 4 weeks
Engine: Android Studio
Language: Kotlin
Created for: Digipen Mobile Programming
Game Description
Its so Dover is a 2D mobile endless runner where you try to get to school on time while trying to avoid all the obstacles from blocking your path. Play as yourself as you can import your face onto the player character to actually feel like you are late to class!
Project Role
My role was the gameplay programmer where I created the gameplay architecture for the game and various gameplay systems in the game. The architecture was the game objects and component system. For the various gameplay systems, I created the obstacle spawning, the win lose condition and the overall game loop.
