Year vs Dog (2018)
Role: Gameplay Programmer
Team Size: 3
Project Length: 3 days
Engine: Unity
Language: C#
Created for: Temasek Polytechnic Game Jam 2018
Game Description
Year vs Dog is a 2D arcade game where you play as a Dog and you need to defeat the evil Nian from ruining Chinese New Year. This game was made during Temasek Polytechnic Game Jam in 2018 where my team won first place in the game jam.
Project Role
My main role was the gameplay programmer where I created the boss AI using a state machine and his attack patterns.
